Admin Notes

How to add files to the library

Keep all files organized by section. Upload PDFs and documents into the matching folder, then update the page link.

1

Upload the file

Place the PDF or document in the correct folder inside the main documents folder.

2

Copy the path

Use the matching relative path, such as ../documents/nextstep-how-to/file-name.pdf.

3

Update the link

Replace the placeholder text or empty resource card with a working document link.

i

GitHub folder reminder

GitHub does not keep empty folders. To create a folder before uploading PDFs, create a placeholder file named .gitkeep inside that folder.
